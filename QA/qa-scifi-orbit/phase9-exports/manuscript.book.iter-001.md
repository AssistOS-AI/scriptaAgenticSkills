# Orbit of the Unreturned

A memory architect learns that her orbital station stays peaceful by deleting politically inconvenient memories.

## Chapter 01 — First Fracture

Cassia Voss reaches the cryo archive ring, where the regulated air hums with memory systems that promise order at a human cost. The environment treats citizens like variables inside a stability model. Riven Arden reads the room too carefully, while Corin Sable keeps alive the version of events other people would rather believe.
In the cryo archive ring, a system alert exposes human fragility inside the station. Cassia Voss tries to restore the corrupted memory trace before governance seals it, but the station treats uncertainty as a security breach instead of a civic fact. Freedom and safety now require conflicting system states. An adaptive signal relay proves the anomaly was administered, not accidental.
"This is not corruption in the archive; it is policy pretending to be repair." Riven Arden says. "If I can isolate the drift, I can still restore the citizens without collapsing the station." Cassia Voss says.
Later, in Orbital Station Horizon, one corrupted memory trace disturbs the engineered calm. Cassia Voss tries to keep human consent visible inside the system response, but the station treats uncertainty as a security breach instead of a civic fact. Memory repair threatens the peace built on forgetting. An adaptive signal relay proves the anomaly was administered, not accidental.
"The station does not fear error; it fears uncurated memory." Cassia Voss says. "Then we stop asking permission from the model that harmed them." Riven Arden says.
Later, in the restricted signal garden, the controlled environment reveals its own fear of freedom. Cassia Voss tries to translate anomaly into public knowledge before prediction closes over it, but the station treats uncertainty as a security breach instead of a civic fact. Human consent collides with predictive control. An adaptive signal relay restores a memory sequence the station was designed to suppress.
"This is not corruption in the archive; it is policy pretending to be repair." Riven Arden says. "If I can isolate the drift, I can still restore the citizens without collapsing the station." Cassia Voss says.
Nothing that has opened here will close easily. No one can yet tell whether restored memory will liberate the station or fracture it.
